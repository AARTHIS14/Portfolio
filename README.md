# Personal Portfolio Website

This is a simple and responsive **personal portfolio website** created using **HTML and CSS**.
It showcases my profile, skills, certifications, projects, and contact details in a clean layout.

---

## About the Project

The portfolio is designed to:

* Introduce myself as a Computer Science Engineering student
* Highlight my technical skills and certifications
* Provide easy access to my LinkedIn and GitHub profiles
* Serve as a basic personal website for academic and career purposes

---

## Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout design

---

## Project Structure

```
Portfolio/
│
├── index.html     # Main HTML file
├── style.css      # CSS styling file
└── README.md      # Project documentation
```

---

## Features

* Clean and minimal UI
* Separate sections for About, Skills, Certifications, Projects, and Contact
* Responsive layout for different screen sizes
* Clickable LinkedIn and GitHub links
* Simple and beginner-friendly code
